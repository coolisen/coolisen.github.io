We are constantly bathed in a sea of particles of radiation but have no mechanism naturally to detect it. So it goes essentially unnoticed. But using only computer parts and stuff from the dollar store, it‚Äôs possible to build a machine that can turn this invisible radiation visible and let us see particles come streaming off of radioactive material. That device is a cloud chamber.
