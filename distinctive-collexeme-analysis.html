This video explains how to run a distinctive collexeme analysis. This is a beginner’s guide. All you need is a concordance program (such as AntConc) and a spreadsheet software (such as Excel). If you want to follow the step-by-step instructions in the video, download the following materials:

AntConc: https://www.laurenceanthony.net/software/antconc/
A concordance of English attributive adjectives: https://drive.google.com/file/d/1y5JXmoJ79YX3wZfoyNMo9xuRPTI7QFqm/view?usp=sharing
A concordance of English predicative adjectives:
https://drive.google.com/file/d/1utUnHQy4hJVeNZDUmQtnFQMGsMJTLhia/view?usp=sharing
