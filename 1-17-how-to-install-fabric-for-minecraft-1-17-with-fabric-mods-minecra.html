With the release of Minecraft 1.17 we also immediately got a new Fabric 1.17 update! In this video I will show you how to download and how to install Fabric for Minecraft 1.17 as well as how to install Fabric mods for Minecraft 1.17! On top of showing you how to install Fabric 1.17 I will also show you how to install Fabric API 1.17, which is required to use some Fabric mods.
