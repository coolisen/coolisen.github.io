1. Windows Firewall
2. Windows Defender
3. Setting up device encryption (BitLocker)
There are several other security measures on Windows 10 which I will not be able to cover in details in this video, for instance,
1. The Data Execution Prevention (DEP) https://support.microsoft.com/en-us/help/875352/a-detailed-description-of-the-data-execution-prevention-dep-feature-in

2. Windows Defender SmartScreen
3. Using Group Policy settings https://www.windowscentral.com/how-apply-local-group-policy-settings-specific-users-windows-10

4. Address Space Layout Randomization (ASLR)
